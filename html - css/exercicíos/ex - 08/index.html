Principais formatações

Negrito / Destaque

Nessa frase termo em negrito utilizando a tag <b> (a tag b não é semântica).

Nessa frase termo em negrito utilizando a tag <strong> (a tag strong é semântica).

Itálico / Ênfase

Nesta frase, temos um termo itálico usando a tag <I>( a tag <I> não é semântica).

Nesta frase, temos um termo em ênfase(itálico) usando a tag <em>(A tag <em> é semântica).

Podemos também criar um texto marcado utilizando a tag <mark>

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois
