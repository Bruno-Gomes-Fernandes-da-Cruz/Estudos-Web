    Lista de Definições

    Serve como um dicionário, tem um termo e abaixo é explicado o seu significado. Se abre com a tag <dl>, o termo abre com a <dt>e o significado pela <dd>.
    Ex:

    HTML
    Linguagem de marcação de hyper texto utilizada para criar conteúdo de site na rede WWW.
    CSS
    Linguagem de marcação utilizada para a especificação de estilos em sites.
    JavaScript
    Linguagem de programação usada para criar interatividade em sites.
