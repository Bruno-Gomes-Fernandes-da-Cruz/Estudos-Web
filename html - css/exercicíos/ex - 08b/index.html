Código-Fonte / Pré-formatação

O comando print("Hello, World") é da linguagem de programação python.
A tag usada para deixar o código python descrito acima em um formato mono-espaçado(todas as letras ocupam a mesma largura) foi a tag <code>

Todavia, a tag code não deixa a código organizado(identado) que nem na IDE. Logo, utilizamos a tag <pre> para deixar o código(texto) presente no site exatamente da forma que está na IDE(VScode). EX:


        numb = input("Digite um número: ")
            if (numb % 2) == 0:
                print(f"O número {numb} é par.)
            else:
                print(f"O número {numb} é ímpar")
        print("Fim do programa!")

Para fazer o bloco de código acima foram juntados as tag <pre> + <code>. Tags unidas ficam mais fortes!!
O <pre> tem que ser posto fora de um parágrafo!

Texto Invertido

Este texto está invertido para a esquerda.
Este texto está invertido para a direita.

Isso é bem inútil, mas o Guanabara quis mostrar do mesmo jeito. A tag usada para inverter foi a tag <bdo>, ela serve para inverter a frase ou palavra envolvida por ela. Dentro desta tag existe o parâmetro dir = "", nele você pode botar a direção rtl(right to left) que deixa a frase invertida ou ltr (left to right) que deixa a frase para a direção normal.
