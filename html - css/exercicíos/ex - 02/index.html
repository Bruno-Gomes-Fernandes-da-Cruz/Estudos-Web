Parágrafos e quebras de linha

Não importa o jeito que você escreve um parágrafo, ele não quebra desse jeito. Além disso, você pode escrever um parágrafo de qualquer jeito
é só botar entre as tags <p> </p> por exemplo:

Este é um parágrafo

tbm

Todos essas palavras acima são parágrafos que estão dentro de um outro parágrafo, ou seja, não tem muitas restrições sobre como botar as tags.br
